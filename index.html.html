About This Project

A rigorous end-to-end machine learning study in R

This portfolio documents a complete machine learning curriculum implemented entirely in R, covering everything from fundamental probability theory to advanced ensemble methods.

Each section applies a different methodology to real-world datasets, with a focus on model evaluation, hyperparameter tuning, and understanding the mathematical foundations behind each algorithm.

Topics covered include classification, regression, dimensionality reduction (PCA/SVD), regularization, bootstrap methods, and ensemble learning.

Key tools: caret, tidyverse, dslabs, randomForest, rpart, MASS — all models validated with proper train/test splits and cross-validation.

Algorithms Used

10+ machine learning methods implemented and evaluated

📈
Logistic Regression
Linear boundary classification with probability outputs. Used across Titanic and BRCA datasets.
🌲
Random Forest
Bagged decision trees with mtry tuning. Best model for Titanic (88.3%). Key variable: area_worst.
🔵
K-Nearest Neighbors
Distance-based classifier. Best model for BRCA (97.4% at k=9). Sensitive to scale — requires normalization.
〰️
LOESS / GAM
Local polynomial regression. Non-parametric smoother used for continuous prediction tasks.
🌳
Decision Trees
Interpretable splits using rpart. Tuned via complexity parameter cp. Used for Titanic and tissue data.
📊
LDA / QDA
Linear and quadratic discriminant analysis. Fast, parametric classifiers assuming Gaussian distributions.
🧮
PCA / SVD
Dimensionality reduction. 3 components explain 98.8% of grade variance. PC1 separates cancer types clearly.
🗳️
Ensemble (Majority Vote)
Combined predictions from multiple models. Achieved 96.5% on BRCA — balancing bias and variance.
05

Key Findings

Insights from applying ML theory to real data

97.4%
KNN Dominates on BRCA
After proper scaling with sweep(), KNN at k=9 outperformed all other models on the breast cancer dataset, including the ensemble.
λ=135
Regularization Eliminates Bias
The "small school effect" — where small schools appear top-ranked due to high variance — is corrected by regularization with optimal λ=135.
98.8%
SVD Reveals Data Structure
Only 3 singular value components capture 98.8% of variance in student grades. Component 1 = overall ability, 2 = arts vs. STEM, 3 = math vs. science.
14.2×
Positive Test Increases Risk
Using Bayes' theorem on a disease with 2% prevalence: a positive test (85% sensitivity, 90% specificity) increases disease probability by 7.4×.
75%
Data Leakage Warning
Selecting predictors using the full dataset before cross-validation artificially inflated accuracy to 75% on completely random data — a textbook leakage case.
PC1
PCA Separates Cancer Types
The first principal component alone clearly separates malignant from benign tumors in the BRCA dataset, with no IQR overlap between groups.
